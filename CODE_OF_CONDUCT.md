# Code of Conduct

## Vores løfte

MieleLogic er et åbent og venligt fællesskab. Vi ønsker at alle — uanset erfaring, baggrund eller nationalitet — føler sig velkomne til at bidrage.

## Forventet adfærd

- Vær venlig og respektfuld over for andre
- Accepter konstruktiv kritik med åbenhed
- Fokuser på hvad der er bedst for fællesskabet
- Vis empati over for andre bidragydere

## Uacceptabel adfærd

- Personlige angreb eller nedladende kommentarer
- Trolling, hån eller provokerende adfærd
- Offentliggørelse af andres private information
- Spam eller irrelevant indhold

## Håndhævelse

Overtrædelser kan rapporteres ved at kontakte projektejer via GitHub. Alle henvendelser behandles fortroligt.

Ved overtrædelse forbeholder vi os ret til at fjerne kommentarer, lukke issues eller blokere brugere.

## Inspiration

Denne Code of Conduct er inspireret af [Contributor Covenant](https://www.contributor-covenant.org/).
